# TBP

TBP is a browser-based, offline text editor.

Requires nodejs and browser that [supports css grid](http://caniuse.com/#feat=css-grid).

To install: 
```
    npm install
```
    
To run: 
```
    node server.js [optional port]
```

Then visit ```localhost:3000``` (or specified port).
